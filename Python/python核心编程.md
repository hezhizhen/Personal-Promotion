# 第2章 列表和元组

数据结构：通过某种方式组织在一起的数据元素的集合（数据元素可以是数字、字符、其他数据结构）

最基本的数据结构：序列（每个元素被分配一个序号，即元素的位置，称为 **索引**，从0开始）（序列中最后一个元素标记为-1，倒数第二个元素为-2，以此类推）

## 2.1 序列概览

python包含6种内置序列，最常用的为 **列表**、**元组**，其他还有：字符串、Unicode字符串、buffer对象、xrange对象

列表和元组的主要区别：列表 **可以** 修改，元组不能

列表中的各元素通过逗号分隔，写在方括号中

序列中可以包含其他的序列

另一种数据结构：容器（包含其他对象的任意对象）

两类主要的容器：序列（每个元素有自己的编号）、映射（每个元素有一个名字/键）

集合：既非 **序列** 亦非 **映射** 的容器类型

## 2.2 通用序列操作

- 索引
- 分片
- 加
- 乘
- 成员资格：检查某个元素是否属于序列的成员
- 迭代

python还有：计算序列长度、找出最大元素和最小元素

### 2.2.1 索引

字符串就是一个由字符组成的序列

序列中的元素可以通过编号分别访问（通过索引获取元素）

字符串字面值能够直接使用索引，如: `'Hello'[1]`

如果一个函数调用返回一个序列，可以直接对返回结果进行索引

### 2.2.2 分片

- 索引：访问单个元素
- 分片：访问一定范围内的元素

分片通过冒号相隔的两个索引来实现，左边索引的元素包含在分片内，右边索引的元素则否

若分片部分包括序列结尾的元素，则置空最后一个索引，如: `numbers[-3:]`

分片需要指定：开始点、结束点、步长(默认隐式设置为1)

步长不能为0；为负数时从右往左提取元素

开始点的元素包括在结果之中，结束点的元素不在分片之中；使用负数步长时，必须让开始点大于结束点

### 2.2.3 序列相加

两种同类型的序列才能进行连接操作

### 2.2.4 乘法

用数字x乘以一个序列，得到的新序列为原来的序列重复x次

空列表：`[]`

None: 内置值，表示“什么都没有”

### 2.2.5 成员资格

使用in运算符检查一个值是否在序列中: 条件为真，返回True；条件为假，返回False （布尔运算符）

### 2.2.6 长度、最小值和最大值

内置函数len、min、max：len返回序列中所包含元素的数量，min返回序列中最小的元素，max返回序列中最大的元素

## 2.3 列表：Python的“苦力”

列表是可变的

### 2.3.1 list函数

字符串不能像列表一样被修改，有时根据字符串创建列表会很有用

list适用于所有类型的序列

### 2.3.2 基本的列表操作

列表可以使用所有序列的标准操作（索引、分片、连接、乘法）

改变列表的方法：元素赋值、元素删除、分片赋值、列表方法

- 元素赋值：使用索引为某个元素赋值（不能为位置不存在的元素赋值）
- 元素删除：使用del语句
- 分片赋值：长度可以不相等，插入、替换、删除

### 2.3.3 列表方法

方法：一个与某些对象有紧密联系的函数，对象可能是列表、数字、字符串、其他类型的对象

方法的调用：`对象.方法(参数)`

列表的方法：(是否有返回值可用`print`查看)

- append：在列表末尾追加新的对象(直接修改原来的列表)
- count: 统计某个元素在列表中出现的次数
- extend: 在列表的末尾追加另一个序列的多个值(直接修改原来的列表，连接操作`+`返回全新的列表)
- index: 从列表中找出某个值第一个匹配项的索引位置(没有则引发异常)
- insert: 将对象插入到列表中
- pop: 移出列表中一个元素（默认是最后一个），并且返回该元素的值(使用pop可以实现栈，用append代替入栈)
- remove: 移出列表中某个值的第一个匹配项
- reverse: 将列表中的元素反向存放
- sort: 在原位置对列表进行排序(有两个可选参数：key、reverse，所有元素根据key来排序；reverse是布尔值，指明列表是否需要反向排序)
- 高级排序：通过compare(x,y)的形式自定义比较函数，会在x<y时返回负数，x>y时返回正数，x=y时返回0. 内置cmp函数提供了默认实现方式

## 2.4 元组：不可变序列

元组是序列，不能修改（字符串亦然）

用逗号分隔一些值，自动创建了元组(可以通过圆括号括起来)

空元组可以用两个圆括号表示

只有一个值的元组在创建时必须加个逗号

### 2.4.1 tuple函数

以一个序列作为参数并把它转换为元组

### 2.4.2 基本元组操作

- 创建元组
- 访问元组元素

### 2.4.3 那么，意义何在

- 元组可以在映射（和集合的成员）中当做key使用，列表则不行
- 元组作为很多内置函数和方法的返回值存在

## 2.5 小结

- 序列：数据结构，包含的元素都进行了编号。包括列表、字符串、元组，列表可变，元组和字符串不可变。通过分片操作可以访问序列的一部分，需要两个索引号来指出分片的起始和结束位置
- 成员资格：in可以检查一个值是否存在于序列（或其他容器中）。对字符串用in可以查找子串
- 方法：一些内置类型具有方法，与特定值联系密切

# 第3章 使用字符串

## 3.1 基本字符串操作

标准的序列操作：索引、分片、乘法、判断成员资格、求长度、最小值、最大值

字符串都是不可变的

## 3.2 字符串格式化：精简版

使用字符串格式化操作符`%`来实现格式化（也可以用作模运算操作符）

在`%`左侧放置格式化字符串，右侧放置希望格式化的值

只有元组和字典可以格式化一个以上的值

`%s`称为转换说明符，标记了需要插入转换值的位置，s表示值会被格式化为字符串

若要在格式化字符串中包括百分号，则使用`%%`

格式化浮点数：f说明符类型，精度表示为一个句点再加希望保留的小数位数
